# 题境随行单机精简版

一个基于Electron+Vue.js开发的Windows桌面刷题应用，支持Excel题库导入导出、多种刷题模式、错题管理和数据统计功能。

## 技术栈

- **前端框架**: Vue.js 3 + Element Plus
- **桌面框架**: Electron
- **数据存储**: JSON
- **Excel处理**: xlsx (js-xlsx)
- **状态管理**: Pinia
- **构建工具**: Vite

## 功能特性

### 1. 题库管理
- 创建、编辑、删除题库
- 支持题库分类
- 题目数量统计

### 2. Excel导入导出
- 标准模板下载
- Excel文件解析与验证
- 批量导入题目
- 题库/错题本数据导出
- 完整数据备份

### 3. 刷题功能
- **顺序刷题**: 按题库原始顺序逐题练习
- **随机刷题**: 从题库中随机抽取指定数量题目
- **模拟考试**: 限时考试，自动计分
- **错题重刷**: 专门针对错题本进行练习

### 4. 答题辅助
- 单题答题耗时统计
- 题目标记（不确定、需复习）
- 跳过题目功能
- 答案提交后显示解析

### 5. 错题本管理
- 自动记录答错题目
- 按知识点筛选查看
- 错题详情展示
- 手动移除错题
- 答错次数统计

### 6. 数据统计
- 整体正确率计算
- 各题型答题情况统计
- 知识点掌握度分析
- 薄弱知识点识别
- 学习时长统计

### 7. 应用设置
- 自定义数据存储路径
- 调整界面字体大小
- 数据备份与恢复

## 安装依赖

```bash
cd Questscape-Pro
npm install
```

## 开发模式

```bash
# 启动开发服务器
npm run dev
```
## 生产构建

```bash
# 构建前端资源
npm run build:renderer

# 打包Electron应用
npm run build:main
```

或者直接运行：
```bash
npm run build
```

打包后的文件在 `dist` 目录中。


## Excel模板格式

### 题目格式

| 题库名称 | | | | | | | | |
|-----------|---|---|---|---|---|---|---|
| **题目内容** | **题型** | **选项A** | **选项B** | **选项C** | **选项D** | **答案** | **解析** | **知识点** | **难度** |
| 1+1=? | 单选题 | 1 | 2 | 3 | 4 | B | 加法基础 | 数学 | 简单 |

### 题型说明
- 单选题：答案填 A/B/C/D
- 多选题：答案填 "AB"（多个字母组合）
- 判断题：答案填 A(正确)/B(错误)
- 填空题：答案填具体内容

### 必填字段
- 题目内容
- 题型
- 答案

## 常见问题

### 导入Excel失败
- 确保Excel文件格式正确
- 检查必填字段是否全部填写
- 验证题型是否为：单选题/多选题/判断题/填空题
